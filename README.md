# Dual Port RAM - Verilog

## Description
This project implements a Dual Port RAM using Verilog HDL.

## Features
- Separate read and write ports
- Synchronous design
- 8-bit data width
- 16 memory locations

## Files
- dual_port_ram.v → Design
- dual_tb.v → Testbench
- waveform.png → Simulation output

##  Simulation
Simulated using Xilinx Vivado.

##  Output
Waveforms show correct read and write operations.
